# LED Indicator PCB

A simple PCB project created in Fusion 360 / Eagle.  
It contains:
- Input & Output headers
- Capacitors (C1, C2, C3)
- Diodes (D1, D2)
- Power indicator LED

## Files
- PCB.f3d → Fusion 360 3D PCB model  
- PCB.brd → Board layout  
- Schematic.fsch → Circuit schematic
